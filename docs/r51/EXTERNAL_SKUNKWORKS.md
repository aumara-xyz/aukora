# R51 external skunkworks — qualification record

Status: **EXTERNAL_SIMULATION · NOT_CANONICAL_RUNTIME · REPRODUCTION_PENDING**

This record publishes the safe scientific content of the R51 Kimi K3 experiments without
promoting an external harness into a canonical Aukora capability. The raw `/mnt/agents/output/dojo`
artifacts were not available in this repository checkout for independent replay. At least one raw
external harness contained an API credential and is deliberately excluded from Git history.

## Memory/index prototype

An external prototype exercised a 12-record content-addressed fixture, consent labels, a tombstone,
and an index rebuild from 93 to 149 entries.

This demonstrates an experiment design. It does **not** establish a 60.2% improvement in recall,
accuracy, latency, or utility: index-entry growth is not a quality metric. The supplied query summary
was internally inconsistent about query count and therefore no recall percentage is claimed here.
Canonical KIRA memory and governed-forgetting law already live in `packages/memory`; self-analysis
and index self-optimization are not current runtime capabilities.

## Constitutional-evolution simulation

The external deterministic harness reported processing **359,427 simulated candidate records** over
20 generations and **381,589 simulated advisory reviews**. It observed zero violations of a
hardcoded `grantsAuthority = false` invariant.

That is a useful stress test of invariant preservation inside that harness. It is not evidence of
359,427 autonomous organisms, emergent traits outside the programmed trait space, or a mathematical
proof that Aukora is unbreakable. Canonical reproduction requires the complete source, exact seed,
environment, raw bundle, independent verifier, mutation tests, and a pin to an exact Aukora commit.

## Live-model experiment boundary

Separate external experiments called **Fugu Ultra** through OpenRouter. Fugu is not Inkling. Those
results are not committed here because the supplied raw harness requires credential scrubbing and
independent reproduction. The canonical Inkling/Nebius manifest remains parked and disabled.

## Admission law

External evidence may enter canonical Aukora only after:

1. secret and private-data scans pass;
2. source, model, environment, seed, and dependency versions are pinned;
3. the evidence bundle and verifier are complete;
4. an independent run reproduces the result at an exact main SHA;
5. claims distinguish observation, simulation, live local proof, and deployed capability;
6. every model/cell remains `advisoryOnly:true` and `grantsAuthority:false`.

This is how the skunkworks stays public without turning spectacle into a false claim.
